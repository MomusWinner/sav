# Description

**SAV** is a desktop tool designed to visualize the output of the **Svace** static analysis tool. It parses Svace reports (CSV format) and creates an interactive graphical representation, helping developers quickly explore and triage static analysis warnings.

Built with the **Odin** programming language and the **VE** engine.
